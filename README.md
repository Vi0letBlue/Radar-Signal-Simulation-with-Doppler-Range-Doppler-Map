
# Radar Simulation with Doppler & Range-Doppler Map

## Description
FMCW Radar simulation:
- C++ generates TX/RX signals with Doppler effect
- Python scripts visualize signals and compute 2D Range-Doppler Map

## Usage
1. Compile & run `C++/main.cpp` → generates `data/radar_data.csv`
2. Run `Python/radar_plot.py` → TX/RX plot
3. Run `Python/range_doppler.py` → Range-Doppler Map

## Images
- `images/tx_rx_plot.png`
- `images/range_doppler_map.png`

- C++ compiler (g++ / Visual Studio)
- Python 3.x
- Python packages: numpy, pandas, matplotlib
